## Usage

```tsx
import React from 'react';

import {List} from './components';

const Example = () => {
  return (
    <List>
      <List.Item
        title="list item1"
        description="list item description here"
        left={() => <List.Icon icon="heart" />}
        right={() => <List.Icon icon="alert-circle" />}
      />
      <List.Item
        title="list item2"
        description="list item description here"
        left={() => <List.Icon icon="heart" />}
        right={() => <List.Icon icon="alert-circle" />}
      />
    </List>
  );
};

export default Example;
```

## Props

### title(required)

Type: `'string'`

Title text for the list accordion.

### description

Type: `'string'`

description text for the list accordion.

### left

Type: `'() => React.ReactNode'`

### right

Type: `'() => React.ReactNode'`

### titleVariant

Type: `'string'`

varinat props from the base text component. <Link to="/docs/components/text">Check all varaints</Link> .

### titleColor

Type: `'string'`

color props from the base text component. The color should match a name defined in the [`theme's color`](https://github.com/IbrahimXplorer/shadow-ui/blob/main/src/theme/colors.ts) object.

### divider

Type: `boolean`

A horizontal separator at the bottom of the list
